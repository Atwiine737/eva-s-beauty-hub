## KEY FLOW EXPLANATION

| Flow | Description |
|------|-------------|
| **Customer Flow** | Browse → Add to Cart → Checkout → Payment → Confirmation |
| **Order Tracking** | Enter Order ID → Search → Display Status |
| **Payment Flow** | Select Method → Complete Payment → Confirmation |
| **Admin Flow** | Receive Order → Process → Update Status |
| **Navigation Flow** | Header → Sections → Footer (with modals) |
